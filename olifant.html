Olifanten weetjes

Deze website is de eerste codeeropdracht die ik op CMD heb gemaakt. De opdracht was om aan de hand van HTML en CSS een website te maken over een onderwerp naar keuze. Ik had gekozen voor olifanten omdat ik het mooie dieren vind. Je kan wel duidelijk zien dat dit mijn eerste website is omdat het er allemaal redelijk basic uitziet en werkt. Ik vond het toen erg spannend om te maken omdat ik nog nooit had gecodeerd. Ik vind het belangrijk om deze opdracht te laten zien omdat dit voor mij een prestatie was. Ik heb, net zo als veel van mijn medestudenten, hiervoor nog nooit gecodeerd. Hoewel ik de website nu anders zou vormgeven en wat dingen zou aanpassen in de code, ben ik nog steeds trots op de website.
